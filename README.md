##SYS-RESER-CANCHA-FRONT

Este repositorio contiene el código Front-end para el Sistema de Reservas de Canchas. Es una **Single Page Application (SPA)** implementada completamente con **Vanilla JavaScript** (ES Modules) y un Router basado en la API History del navegador.

## Requisitos e Instalación

Para ejecutar este proyecto, necesitas tener **Node.js** (versión 16+) y **Git** instalados en tu sistema.

### 1. Obtener el Código

Clona el repositorio desde GitHub y navega a la carpeta:

[git clone [https://github.com/mmenaaguilar/sys-res-cancha-front.git](https://github.com/mmenaaguilar/sys-res-cancha-front.git)](https://github.com/mmenaaguilar/sys-res-cancha-front.git)
cd sys-res-cancha-front

### 1. Inicializar el proyecto

Clona el repositorio desde GitHub y navega a la carpeta:

npm install

### Ejecución Local (Development)

npm start
